步骤一
首先自己弄一个有密码的压缩文件
压缩1.txt为1.rar 并设置密码为456
C:\Progra~1\WinRAR\WinRAR.exe a 1.rar 1.txt -p456

步骤二
生成1.rar(带密码)后
为了解密需生成一个字典文件
@echo off
for /l %%1 in (0,1,9) do (
     for /l %%2 in (0,1,9) do (
        for /l %%3 in (0,1,9) do (
        echo %%1%%2%%3>>密码字典文件.txt
        )
     )
)
保存为*.bat让其运行生成字典文件

步骤三
将下面代码保存为“压缩文件密码破解.bat”输入路径运行 就能把密码跑出来了！

@echo off
title 压缩文件密码破解   BY-qp    QQ179157977
color 2e
::设置要解密的压缩文件(路径不允许有空格)
set /p 文件=       压缩文件路径:
::设置字典文件的位置
set /p 字典=       字典文件路径:
::建立文件夹，将解压出来的文件放到里面去
md 解压结果
cls                                    
for /f "delims=" %%a in ('type "%字典%"') do (
     set mima=%%a
     echo                  正在用 %%a 作为密码破解
     C:\Progra~1\WinRAR\WinRAR.exe x %文件% * 解压结果 -p%%a y -ibck -inul
     dir /a-d/b/s 解压结果\*.* | find /v "找不到文件" && goto win
     echo                                             %%a 不是压缩文件的密码
     echo %%a 不是压缩文件的密码>txt.txt
)
echo 字典文件包含的密码都不是压缩文件的密码>txt.txt
echo 字典文件包含的密码都不是压缩文件的密码
exit

:win
cls
echo 密码为 %mima%
echo 密码为 %mima%>txt.txt
pause

步骤四
让“压缩文件密码破解.bat”跑出密码来吧
