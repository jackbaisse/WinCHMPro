1 、在D盘手动建立文件夹exbak
2、 查询oracle中建立的文件夹
select * from dba_directories;
3 创建文件夹 如果没有的话
create directory mypump as 'D:\bakfile';
4 将文件夹权限赋值给用户
grant read, write on directory mypump to yidandba;
5 导出文件，注意数据泵导出的dmp不和一般导入导出
导出方式：
1、“full=y”，全量导出数据库
expdp sys/oracle@orcl dumpfile=expdp.dmp directory=mypump full=y logfile=expdp.log
2、schemas按用户导出
expdp user/passwd@orcl schemas=user dumpfile=expdp.dmp directory=mypump  logfile=expdp.log
3、按表空间导出
expdp sys/passwd@orcl tablespace=tbs1,tbs2 dumpfile=expdp.dmp directory=mypump  logfile=expdp.log
4、导出表
expdp user/passwd@orcl tables=table1,table2 dumpfile=expdp.dmp directory=mypump  logfile=expdp.log
5、按查询条件导出
expdp user/passwd@orcl tables=table1='where number=1234' dumpfile=expdp.dmp directory=mypump  logfile=expdp.log
