1 、在D盘手动建立文件夹exbak 


2、 查询oracle中建立的文件夹
select * from dba_directories; 


3 创建文件夹 如果没有的话
create directory mypump as 'D:\bakfile';

4 将文件夹权限赋值给用户
grant read, write on directory mypump to yidandba;

5 导出文件，注意数据泵导出的dmp不和一般导入导出
expdp nydba /sa@emr   directory=mypump dumpfile= emrpj.dmp  logfile= emrpj.log full= y

注意：
 1-4语句只要执行一次，以后不需要执行
