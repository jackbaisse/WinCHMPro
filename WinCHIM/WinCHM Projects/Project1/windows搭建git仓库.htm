1、下载工具包jdk-8u、gitblit1.8.0、git-2.17.0-64位

2、搭建java环境、配置环境变量

3、安装gitblit工具
1)修改gitblit安装目录下的data文件夹下的gitblit.properties文件
将其中的include = defaults.properties这一行注释掉，添加include =my.properties
2）在data文件夹下找到defaults.properties文件，这个是gitblit默认的配置文件，我们复制一份该文件，重命名为my.properties文件。然后在安装目录下新建一个文件夹project，作为代码仓库目录。

4、修改my.properties文件，设置本地仓库配置。
4.1git.repositoriesFolder（代码仓库路径）：这里修改为自己的仓库目录路径，不修改则默认为gitblit目录下的git目录。
4.2server.httpPort: 设定http协议的端口号，后面登陆会用到。默认0表示禁用http协议。
4.3server.httpBindInterface: 设定服务器的IP地址，这里我们就设定地址为电脑的当前IP

5、运行gitblit根目录下的gitblit.cmd批处理文件，启动git服务器。

6、6.git服务器启动后，在浏览器中输入设置的IP:端口号，出现如下界面，即说明服务器启动成功

7、7.然后登陆代码仓库，初始默认为用户名：admin，密码：admin

8、在添加文件和目录的时候可能会遇到一个问题，就是回车符问题。如果遇到了，可以尝试执行以下命令解决。
git rm -r --cached ./ 2 git config core.autocrlf false 3 git add ./
