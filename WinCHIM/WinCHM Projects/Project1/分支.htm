1、创建分支、切换到分支
git checkout -b dev  --新版git switch -c <name>

git checkout命令加上-b参数表示创建并切换，相当于以下两条命令：

git branch dev   --branch 分支

git checkout dev --checkout 检出

2、查看当前分支

git branch --列出所有分支，前面分支会标*号

git add filename.txt
git commit -m "mark"

3、切换主分支
git checkout master  --最新版本git switch

4、合并主分支
git merge dev

5、删除分支
git branch -d dev

6、查看分支
git branch
